🌟 תכונות מרכזיות
👑
Dual Super-Admin
שני המשתמשים הראשונים מקבלים אוטומטית הרשאות סופר-מנהל. ניהול מלא של כל המערכת.
📚
סילבוס חכם – 13 שלבים
סילבוס שחייה מובנה עם 72 משימות ב-13 שלבים. תמיכה בהעלאת סילבוס מותאם (CSV) לכל נושא.
✅
נוכחות וקרדיטים
סימון נוכחות מנכה קרדיט אוטומטית. התראה על גרעון בשיעורים פרטיים. שחזור בביטול נוכחות.
💬
צ'אט היררכי ובטוח
תקשורת לפי היררכיה. פילטר שפה אוטומטי (רשימה שחורה). חסימת מדיה. התראות לסופר-מנהל.
📱
דוחות WhatsApp – 6 שפות
יצירת דוחות בעברית, אנגלית, ערבית, רוסית, צרפתית וספרדית. אפשרות המלצת מוצר מהחנות.
✍️
חתימות הורים שנתיות
חתימה דיגיטלית ציורית על מסך המגע. תוקף שנה. מעקב אוטומטי אחר חתימות חסרות.
🏊
שיבוץ נתיבים
מפה גרפית ותפריט רשימה לשיבוץ תלמידים לנתיבים. צבעי רמה לפי התקדמות.
📁
ניהול קבצים מקומי
העלאת קבצים, דפדוף בתיקיות המחשב, ייבוא CSV ישיר לסילבוס ולתלמידים.
🛒
חנות ומלאי
חנות מוצרים (ניהול ע"י סופר-מנהל). מלאי ציוד עם קטגוריות מותאמות והתראות מלאי נמוך.
🚀 התקנה בארבעה שלבים
1
הורד את הקוד
לחץ על "הורד את האפליקציה" למעלה, או שכפל מ-GitHub
git clone https://github.com/israelheber675-prog/SwimTrack-Pro.git
2
התקן Python
יש צורך ב-Python 3.10 ומעלה. הורד מ-python.org
python --version
3
הפעל את start.bat
לחץ פעמיים על start.bat – הוא מתקין הכל ופותח את הדפדפן אוטומטית
start.bat
4
הירשם כסופר-מנהל
שני הנרשמים הראשונים מקבלים סופר-מנהל אוטומטית
http://localhost:8501
